# Molink 功能模块清单

本文档汇总了 Molink 前端项目当前已实现的 **13 个功能模块**，涵盖从用户认证到编辑器核心的完整功能链路。

---

## 1. 用户认证与授权

**核心文件**：`src/components/auth/Login.tsx`、`src/context/AuthContext.tsx`、`src/api/auth.ts`

**功能描述**：
- 用户登录与注册（邮箱 + 密码）
- JWT Bearer Token 认证机制，Token 存储于 `localStorage`
- 访客模式：未登录用户可匿名使用，数据暂存本地
- 登录成功后自动进入工作空间
- 登出功能

---

## 2. 工作空间管理

**核心文件**：`src/api/workspaces.ts`、`src/App.tsx`

**功能描述**：
- 工作空间列表查询
- 首次登录自动创建工作空间（默认名"我的空间"）
- 工作空间作为页面数据的顶层容器

---

## 3. 页面管理

**核心文件**：`src/Sidebar.tsx`、`src/App.tsx`、`src/api/pages.ts`

**功能描述**：
- **页面 CRUD**：创建、更新、删除页面
- **树形层级结构**：支持无限层级的父子页面关系，Sidebar 递归渲染树形目录
- **面包屑导航**：顶部标题栏展示当前页面的祖先链，支持点击跳转
- **前进/后退历史**：类似浏览器的页面导航栈（`backStack` / `forwardStack`）
- **最近页面**：Sidebar 底部展示按 `updatedAt` 排序的最近访问页面
- **页面时间戳**：记录 `createdAt`、`updatedAt`、`createdBy`、`updatedBy`

---

## 4. 块级富文本编辑器

**核心文件**：`src/Editor.tsx`、`src/BlockElement.tsx`、`src/Leaf.tsx`

**功能描述**：
- 基于 **Slate 0.118** 的块级编辑器架构
- 支持 **13 种块类型**：
  - `paragraph`（段落）
  - `heading-one` / `heading-two`（一级/二级标题）
  - `bulleted-list`（无序列表）
  - `numbered-list`（有序列表）
  - `todo`（待办事项，含勾选状态）
  - `toggle-list`（折叠列表）
  - `blockquote`（引用块）
  - `code-block`（代码块）
  - `divider`（分割线）
  - `callout`（高亮提示块）
  - `page-link`（页面链接块）
- **内容同步**：自动将 Slate 内容同步到后端 `blocks` 表
- **块级占位符**：不同块类型有各自的空状态提示，段落仅最后一行显示占位符
- **拼写检查禁用**：`spellCheck={false}` 提升中文输入体验
- **底部留白**：编辑器末尾预留 `33vh` 空白区域，方便滚动到文档底部

---

## 5. 页面装饰

**核心文件**：`src/Editor.tsx`、`src/components/IconPicker.tsx`

**功能描述**：
- **封面图片**：
  - 支持上传封面（登录后走后端文件上传，未登录降级为 Base64）
  - 封面高度响应式（基于视口高度 `COVER_VH`）
  - 封面位置拖拽调整（`object-position` Y 轴百分比）
- **图标选择器**：
  - 支持三种图标类型：Emoji、Lucide 图标（可变色）、自定义图片上传
  - 最近使用记录（`localStorage` 缓存）
  - 搜索筛选功能
  - 点击页面标题旁的图标区域触发选择器

---

## 6. 拖拽与多选交互

**核心文件**：`src/Editor.tsx`、`src/BlockElement.tsx`

**功能描述**：
- **拖拽重排序**：拖拽块左侧的 4 点手柄（2×2 SVG）进行块重排序
- **拖拽幽灵**：拖拽时生成 DOM `cloneNode(true)` 的半透明幽灵元素，宽度与原块完全一致，移除选中态和拖拽手柄
- **框选多选**：在编辑器空白区域拖拽框选，自动计算与块元素的交集，批量设置 `selected: true`
- **多选拖拽指示器**：拖拽已选中块到未选中区域时，显示蓝色指示线；拖拽到已选区域内则隐藏
- **拖拽优化**：拖拽期间纯 DOM `dataset.blockSelected` 操作，`requestAnimationFrame` 节流，鼠标释放后同步回 Slate state
- **拖拽手柄悬停抑制**：块失去焦点后，拖拽手柄自动隐藏，直到鼠标再次移动

---

## 7. 剪贴板复制粘贴

**核心文件**：`src/Editor.tsx`

**功能描述**：
- **全局拦截**：重写 `ReactEditor.setFragmentData`，拦截系统的 copy/cut 行为
- **Markdown 序列化**：选中块复制时，序列化为带前缀的纯文本：
  - `# ` → heading-one
  - `## ` → heading-two
  - `- ` → bulleted-list
  - `- [x] ` / `- [ ] ` → todo（含勾选状态）
  - `>> ` → toggle-list
  - `> ` → blockquote
  - 无前缀 → paragraph
- **智能粘贴解析**：粘贴时根据前缀自动还原为对应块类型
- **5 条插入规则**：
  1. 无焦点 + 无选中 → 插入到文档末尾
  2. 光标在非空块 → 插入到该块下方
  3. 多选状态 → 插入到最低选中非空块下方
  4. 光标在空段落 → 替换该段落
  5. 多选尾部有空段落 → 忽略尾部空段落确定插入点
- **纯文本降级**：非特殊块类型的纯文本按 Slate 默认行为处理

---

## 8. 斜杠命令菜单

**核心文件**：`src/components/SlashCommandMenu.tsx`

**功能描述**：
- 在空段落中输入 `/` 触发命令菜单
- 支持搜索过滤块类型
- 可选块类型包括：标题、列表、待办、折叠、引用、代码块、分割线、页面链接等
- 键盘导航（上下箭头选择，Enter 确认）
- 菜单定位跟随光标所在块

---

## 9. Markdown 快捷输入

**核心文件**：`src/withMarkdownShortcuts.ts`

**功能描述**：
- **快捷键转块类型**：
  - `# ` → heading-one
  - `## ` → heading-two
  - `- ` → bulleted-list
  - `1. ` → numbered-list
  - `- [ ] ` → todo
  - `>> ` → toggle-list
  - `> ` → blockquote
- **特殊 Enter 行为**：
  - 空标题/折叠块中按 Enter → 下方插入新段落，保留当前块
  - 空列表/待办中按 Enter → 转为段落
- **特殊 Backspace 行为**：
  - 非段落块在块首按 Backspace → 转为段落
  - 空段落按 Backspace → 删除该块，光标移至前一块末尾

---

## 10. 页面链接与导航

**核心文件**：`src/BlockElement.tsx`（page-link 类型）

**功能描述**：
- **块级页面引用**：在编辑器中通过斜杠菜单或粘贴插入 `page-link` 块
- **自动同步**：`page-link` 块的 `pageId` 与页面数据实时关联
- **Hover 预览**：鼠标悬停时显示目标页面的图标 + 标题预览卡片，带进入/退出动画
- **点击跳转**：点击页面链接块直接导航到目标页面
- **拖拽手柄**：页面链接块也有独立的拖拽手柄，支持参与拖拽重排序

---

## 11. 设置与个性化

**核心文件**：`src/components/SettingsModal.tsx`、`src/context/ThemeContext.tsx`、`src/components/UserMenu.tsx`

**功能描述**：
- **设置弹窗**：Notion 风格设置面板，左侧导航 + 右侧内容区
  - 账号：档案管理、邮箱、密码、两步验证、设备登出、删除账号
  - 偏好：外观主题、输入选项、语言、时区、桌面应用设置、隐私
  - 通知：邮件通知、浏览器推送、协作通知、提醒
- **主题切换**：日间 / 夜间 / 跟随系统，基于 `ThemeContext`，支持 `resolvedTheme`
- **用户菜单**：固定定位下拉菜单（`fixed` + `getBoundingClientRect` 计算位置），显示用户头像、邮箱、登出入口；未登录状态显示"临时工作空间"
- **动画**：设置弹窗使用 `AnimatedPresence` 实现淡入淡出动画

---

## 12. Landing Page（品牌首页）

**核心文件**：`src/pages/LandingPage.tsx`

**功能描述**：
- 3D 地球动画背景（基于 `cobe`）
- 品牌展示区（`BlurFade`、`TextAnimate` 等动画组件）
- "开始使用"按钮进入访客模式
- "登录"按钮唤起登录弹窗
- 响应式设计，适配移动端

---

## 13. 数据持久化与迁移

**核心文件**：`src/App.tsx`、`src/components/LoadingScreen.tsx`

**功能描述**：
- **localStorage 降级**：未登录用户的所有页面数据自动持久化到 `localStorage`
- **封面位置本地缓存**：封面调整位置存储于 `localStorage`，避免后端不支持时的数据丢失
- **登录后数据迁移**：
  - 检测本地是否有有效数据（有标题或有内容的页面）
  - 弹出迁移确认对话框，用户可选择"导入到云端"或"放弃本地数据"
  - 迁移时逐页创建后端页面 + 块内容
- **加载屏幕**：首次进入应用时的品牌加载动画（`LoadingScreen`）
